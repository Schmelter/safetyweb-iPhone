SafetyWeb API

Host

https://www.safetyweb.com

Required Parameters

api_id
timestamp
signature

Document Notes

Things defined with {} are items that should be replaced with the appropriate data.  {api_key} is provided to you by SafetyWeb.  {method} is provided in this documentation further down in the method definition section. Other items are replaced with data from API responses.

Parameter Definitions

api_id

The api_id is provided to you by SafetyWeb.  It is a 32 character string.

timestamp

Timestamps are UTC timestamps.

signature

The signature is a base64 encoded sha256 using your API Key of the following string:

GET
www.safetyweb.com
/api/v1/uri
alphabetized=by_keys&query=parameters


The code to do this in PHP would be:

$params = array(“api_id” => “12345”, “timestamp” => “2010-01-12T13:59:33Z”);
uksort($params, ‘strnatcmp’);
$sig_str = sprintf(“GET\n%s\n%s\n%s”, “www.safetyweb.com”, “{method}”, http_build_query($params));
$sig = base64_encode(hash_hmac(‘sha256’, $sig_str, “{api_key}”, true));
$params[‘signature’] = $sig;
$url = sprint(‘https://%s%s?%s’, ‘www.safetyweb.com’, ‘{method}’, http_build_query($params));

token

Token’s are a string returned via the login request.  They expire 12 hours after being generated.
